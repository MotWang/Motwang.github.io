## Frame

If autonomous agents become the default surface for commerce, incumbents' moats reprice around three questions: **who owns the routing**, **who owns the transaction rail**, and **who owns post-purchase trust**.

## Routing

The agent picks the merchant. Whoever the agent trusts to enumerate the option set becomes the new "aisle." Right now that layer looks like it will fragment across model providers rather than consolidate under a single retailer.

## Transaction rail

Payment, fulfillment status, and dispute resolution still need a trusted backend. Platforms with embedded logistics and seller-of-record infrastructure keep an edge; pure discovery layers get commoditized.

## Trust

Returns, authenticity, and service quality become agent-evaluated signals rather than star ratings. Brands that can't be verified programmatically lose shelf space in agent-mediated flows.
