# Motwang.github.io

Personal homepage of **Qifeng Wang (汪琪峰)**.

🌐 Live site: [https://motwang.github.io](https://motwang.github.io)

## About

This site is built with [Jekyll](https://jekyllrb.com/) based on the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template.

## Local Preview

**Requires Ruby 3.0+** (macOS system Ruby 2.6 will not work).

```bash
cd /Users/wangqifeng1/Desktop/hit-perfect.github.io-main
bash run_server.sh
```

Open **http://127.0.0.1:4000** in your browser.

If Ruby is too old, `run_server.sh` will print install steps (Homebrew + Ruby 3.1).

**Alternative:** push to GitHub and view at https://motwang.github.io (no local setup needed).
